Consola 2:
(1). sudo useradd paw
(2). sudo passwd paw ;paw es la contraseña Y el usuario
3. su paw
4. psql -d paw4 -U paw -W ;conecta a la base de datos paw4 con el usuario paw y pide contraseña

() se hacen solo 1 vez.

Como creamos la BD:
su postgres
psql 
CREATE DATABASE paw4 WITH ENCODING 'UTF8';
CREATE USER paw PASSWORD 'paw';
GRANT ALL PRIVILAGES ON DATABASE paw4 TO paw;
